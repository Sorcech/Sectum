# Tabs component

<br>

Tabs component is used when there are many options or when the options are uncertain

## Basic usage

```
import Tabs from 'sectionui'
import TabPane from 'sectionui'
```

<Tabs name="123">
  <TabPane name="tab1" tab="tab1">
    tab111
  </TabPane>
  <TabPane name="tab2" tab="tab2">
    tab222
  </TabPane>
  <TabPane name="tab3" tab="tab3">
    tab333
  </TabPane>
</Tabs>
